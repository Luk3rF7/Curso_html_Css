Lista

    list-style: define como vai demostrar a lista exemplo

  • none = sem marcação
  • square = quadrado
  • decimal = numero
  • outisde = bolinhas
  • circle = circo
  • text-align vai posiciona o texto ou que tiver dentro do elemento

Link

    utilizamos nos link o text decorate

  • text-decoration: none vai remover linha do link ou texto
  • dashed vai deixa linha pontilhada

pseudo-Classe

    hover

  • hover vai dar efeito qndo passamo o mouse ou evento
  • visited ele modifica mostrando que visitamo aquele link ou page

Position

    Position e reponsavel pela posição do elemento

  • static valor padrao segue fluxo da pagina top,bottom,rifght e left nao funciona
  • relative elemeto e posicionado em relaçção ao normal proriedade top,bottom,right,left são aplicaveis ela movem exemple: .realtive{
    position:relative;
    left: 30px;
    top:50px; ele vai deslocar 50xp
    border:3px solid black;
    }

  • absolute

    elemento e posicionado em relação ao seu elemento pai mais proximo
    que possui position:relative,
    se o elemento nao tiver nenhum elemento pai
    position relative ,posicionamento em relação ao body

    Exemplo

    div position:relative
    The div postion:absolute

  • Fixed

    Fixed elemento e posicionado em relação ao viewport

    sempre fica na mesma posição ,mesmo se a pagina rodar

    Propriedade top,botton,left,right são aplicaveis

    Exemplo


    elemento position fixed viewport
  • sticky

    elemento e posicionadoem relaçao ao scroll do usuario

    Propriedade top,botton,left,right são aplicaveis
